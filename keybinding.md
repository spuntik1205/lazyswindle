# LazySwindle Keybindings Reference

Here is the default set of keybindings for the LazySwindle window manager. 
**Note:** The `Logo` key is usually your Windows or Super key.

## Application Launchers
- **`Logo + T`**: Open Terminal (Ghostty)
- **`Logo + D`**: Open App Launcher
- **`Logo + Space`** or **`Logo + Comma`**: Open Settings Menu

## Window Management
- **`Logo + Q`**: Close focused window
- **`Logo + Shift + T`**: Toggle floating mode for the focused window
- **`Logo + Shift + F`**: Toggle fullscreen mode

## Niri-style Directional Focus (Switching Windows)
- **`Logo + Arrow Keys`** (Up, Down, Left, Right): Change focus to the window in that direction

## Niri-style Directional Movement (Moving Windows)
- **`Logo + Shift + Arrow Keys`**: Move the focused window in that direction

## Workspace Switching
- **`Logo + 1 through 9`**: Switch to workspace 1-9

## Move Window to Workspace
- **`Logo + Shift + 1 through 9`**: Move the currently focused window to workspace 1-9

## System Controls & Media
- **`Logo + Shift + E`**: Quit compositor (Logout)
- **`Print Screen`**: Screenshot selection to `~/Pictures`
- **`Shift + Print Screen`**: Fullscreen screenshot to `~/Pictures`
- **`Media Keys`**: Volume up/down, mute, and brightness control
