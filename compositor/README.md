## Swindle (LazySwindle Core)

> "The core of the lazy Wayland ecosystem."

This directory contains the core Wayland compositor for the **LazySwindle** ecosystem. It is a fork of the original [Swindle](https://github.com/kantiankant/swindle) (which itself was a fork of dwl).

In LazySwindle, this compositor has been integrated with a live-reloading Lua configuration (`config.lua`) that allows an external shell (like our AGS settings app) to dynamically update gaps, colors, and borders on the fly!

Please see the root `README.md` of this repository for full details, installation instructions, and documentation.
